## 使用するコマンド・ツール

### コマンド代替

| 一般的なコマンド | 使用するコマンド |
|------------------|------------------|
| `grep`           | `rg` (ripgrep)   |
| `find`           | `fd`             |
| `ls`             | `eza`            |
| `npm`            | `pnpm`           |
| `npx`            | `pnpx`           |
| `python3`        | `uv run python3` |

### Git コマンド

- ブランチの作成は `git switch` コマンドを使用してください。
